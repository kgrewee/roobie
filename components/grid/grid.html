Nowrap is the Default

When there isn't enough space in a container for the sub-containers, the wrap property decides what to do. The default flex style is nowrap, so the sub containers are squished to fit on one row.

Outer container -> max-height: 200px

Sub containers -> width: 30% & height: 50px
