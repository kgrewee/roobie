Nowrap is the Default

For this demo, the max height of the grids is 225px. The sub-containers are 30% width & 50px height. There isn't enough space in the container for the sub-containers, so the wrap property decides what to do. The default flex style is nowrap, so the sub-containers are squished to fit on one line for both flx-row & flx-col.
