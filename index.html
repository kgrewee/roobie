Roobie

Reduce css clutter, improve development speed and mobile-responsiveness.

How does it work?

Roobie is a collection of stylesheets that allows clients to build mobile-responsive websites very quickly with minimal css.

Can I use Roobie in my current development environment?

Yes. Roobie is made to be plug and play. Use it only when and where you need it. No extra overhead.

How do I install it?

Download roobie.css or copy it from the github repo and add it to your html.

<link rel="stylesheet" href="roobie.css">
