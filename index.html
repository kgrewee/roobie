Roobie

Start every Web project with CSS Classes and Themes


Install

Roobie is simply a stylesheet.

Copy roobie.css from the github repo and link it to your html.

<link rel="stylesheet" href="roobie.css">

Browse

Roobie offers a variety of pre-built components styled with our roobie.css. Check them out and use them as a starting point.

How does it work?

Roobie focuses on centralizing styles in pre-defined CSS classes to reduce code and improve development speed. Once you know the basics, you can build full websites without writing any CSS.

Advantages

  • Less clutter
  • Less cleanup
  • Consistency
  • Mobile-responsive
  • Starter components
  • Content and color focus, not basic styles
