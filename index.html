Roobie

Start every Web project with CSS Classes and Themes

Install

Roobie is simply a stylesheet.

Copy roobie.css from the github repo and link it to your html.

<link rel="stylesheet" href="roobie.css">

Centralized Styles

Roobie focuses on centralizing styles in pre-defined CSS classes to reduce code and improve development speed. Build full websites with barely any CSS. Change the variables found in roobie.css and use it as a main stylesheet for your Web application.

Components

We offer a variety of pre-built components styled with our CSS classes. Check them out and use them as building blocks.

Advantages

  • Less clutter
  • Less cleanup
  • Consistency
  • Mobile-responsive
  • Starter components
  • Content and color focus, not basic styles
