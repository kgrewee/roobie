Nowrap is the Default

For this demo, the max_eight of the grids is 225px. The sub-containers are 30% width & 50px height. There isn't enough space in the container for the sub-containers, so the flex-wrap property decides what to do. The default Roobie flex-wrap value is nowrap, so the sub-containers are squished to fit on one line.

Wrap

Nowrap is useful, but in this case, we want to wrap our items to keep them the same size. Simply add the wrp class to add wrap to the Grid. Use flex classes to align grid content.
