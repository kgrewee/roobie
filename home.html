Install

Roobie is a Web Framework to build web applications quickly and efficiently. Some JavaScript functions depend on JQuery.

Download roobie.css Download roobie.js

JQuery

<script src="https://code.jquery.com/jquery-3.6.0.min.js" integrity="sha256-/xUj+3OJU5yExlq6GSYGSHk7tPXikynS7ogEvDej/m4=" crossorigin="anonymous"></script>

HTML

<link rel="stylesheet" href="roobie.css"> <script src="roobie.js"></script>

CSS @import "roobie.css";

Advantages

  • Building blocks Start every project 10 feet off the ground, not zero.
  • Less clutter CSS and JavaScript are centralized.
  • Less cleanup Re-using styles and functions makes for less cleanup when removing content.
  • Consistency By using Roobie, you have a standard that can be shared across a team for development.
  • Pre-built components We provide common Web pages and components for anyone to use.
