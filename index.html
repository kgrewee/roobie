Roobie

Reduce css clutter, improve development speed and mobile-responsiveness.



How does it work?

Roobie is a collection of styles that allows clients to build mobile-responsive websites very quickly with minimal css.

Advantages

  • Less clutter
  • Less cleanup
  • Consistency
  • Mobile-responsive
  • Starter components
  • Content and color focus, not basic styles

Can I use Roobie in my current development environment?

Yes. Roobie is made to be plug and play. Use it only when and where you need it. No extra overhead.

How do I install it?

Copy roobie.css from the github repo and link it to your html.

<link rel="stylesheet" href="roobie.css">
