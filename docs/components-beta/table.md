# Table
Easily customizable table styles.

## Basic
<div class="row row-spacer">
    <table class="table">
        <tr>
            <th>Name</th>
            <th>Color</th>
            <th>Description</th>
        </tr>
        <tr>
            <td>George</td>
            <td>Red</td>
            <td>A kind fellow.</td>
        </tr>
        <tr>
            <td>Lisa</td>
            <td>Green</td>
            <td>The best gardener.</td>
        </tr>
        <tr>
            <td>Billy</td>
            <td>Yellow</td>
            <td>Space flight specialist.</td>
        </tr>
    </table>
