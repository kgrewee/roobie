Roobie

Start every Web project with CSS Classes & Themes

Install

Roobie is simply a stylesheet.

Copy roobie.css from the github repo and link it to your html.

<link rel="stylesheet" href="roobie.css">

Centralized Styles

More content, less redundant styles. Roobie focuses on centralizing styles in pre-defined CSS classes to reduce code and improve development speed. Build full websites solely with Roobie CSS. Change the variables and use it as a main stylesheet or plugin for your Web application.

Components

We offer a variety of pre-built components styled with our CSS classes. Check them out and use them as building blocks.
