Navbar

Navbar

nav-top

Navbar at the top of the page. Default height auto. Combine with abs or sticky.

nav-row

Horizontal navbar. Defaults to width 100%, height auto.

nav-col

Vertical navbar. Defaults to height 100%, width auto.

nav-title

Navbar title, usually the site name. Default xl margin and black color.

nav-head

Navbar heading. Default s margin and black color.

nav-item

Navbar item, usually a link. Default s margin and black color.
