# Path
Display the current website path, or a group of items.

## Basic
<ul class="path">
    <li class="path-item">Home</li>
    <li class="path-item">Shopping</li>
    <li class="path-item">Cart</li>
</ul>

```html
<ul class="path">
    <li class="path-item">Home</li>
    <li class="path-item">Shopping</li>
    <li class="path-item">Cart</li>
</ul>
```
