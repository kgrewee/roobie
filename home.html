Install

Roobie is a framework to build web applications quickly and efficiently. Some JavaScript functions rely on JQuery. Basic knowledge of HTML, CSS & JS is required to get the most out of Roobie.

Download roobie.css Download roobie.js

in HTML

<link rel="stylesheet" href="roobie.css"> <script src="https://code.jquery.com/jquery-3.6.0.min.js" integrity="sha256-/xUj+3OJU5yExlq6GSYGSHk7tPXikynS7ogEvDej/m4=" crossorigin="anonymous"></script> <script src="roobie.js"></script>

in CSS @import "roobie.css";

Centralized Styles

More content, less redundant styles. Roobie focuses on centralizing styles in pre-defined CSS classes to reduce code and improve development speed. Build full websites solely with Roobie CSS. Change the variables and use it as a main stylesheet or plugin for your Web application.

Components

We offer a variety of pre-built components styled with our CSS classes. Check them out and use them as building blocks.

Advantages

  • Building blocks. Start every project 10 feet off the ground, not at zero.
  • Less clutter. Styles and JavaScript are centralized.
  • Less cleanup. Re-using styles and functions makes for less cleanup when removing content.
  • Consistency. By using Roobie, you have a standard that can be shared across a team for development.
  • Pre-built components. We provide common Web pages and components for everyone to use.
